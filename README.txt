Game Description:
This game simply asks you to guess a four-unique-digit number within 10 trials.
For example, 1234 will be a possible one but 1123 will not occur in the game.
Every time you enter a number, a hint like XAXB will be returned to you.0A1B 
means you have one number correct but in wrong place. 1A0B means you have one
number correct as well as in the correct place.

Link to Github repo:
https://github.com/yaoxisheng/EECS-481-hwk2

How to build the code(On eclipse):
1.Clone the project folder (called GuessTheNumber) from git repo.
2.Import it in Eclipse: File->import->General->Existing Projects into Workspace
  ->Select the path of folder GuessTheNumber as root directory->Finish
3.Run it on an Emulator. Instructions on creating an AVD(emulator):
  http://developer.android.com/training/basics/firstapp/running-app.html#Emulator
  Please select 5.1" WVGA (480 x 800:mdpi) as the device.
